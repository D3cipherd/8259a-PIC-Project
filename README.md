# 8259a-PIC-Project

## About
 Programmable Interrupt Controller 8259/8259A Implementaion in Verilog. 
## Features
- 8086 Mode
- Fully Nested Mode
- End of Interrupt
- Automatic End of Interrupt(AEOI) Mode
- Automatic Rotation
- Interrupt Masks
- Reading the 8259A Status
- Edge and Level Triggered Modes
- Cascade Mode
